HTML5 Canvas Tutorial - Checker Board

In this tutorial we will create a simple checker board

The first thing we are going to do is add a canvas element to our page and give it an ID

<canvas id="canvas"> </canvas>

This is where our efforts will be displayed

In our script section, we will create a variable to refer to our canvas element

<script>
var canvas = document.getElementById('canvas');
</script>

Next we create another variable that will refer to the canvas API

<script>
var canvas = document.getElementById('canvas');
var ctx = canvas.getContext('2d'); </script>
